Discuss the scope of var, let, and const

var is global scope variable and let ,const are block scope. we can't access this outside of a function or any block but we can access var variable anywhere from our file. If we declare a variable called number inside a function useing var keyword we can access the number variable outside of our function we can redeclure number variable it will not give us any error.


var number = 1;
var number = 2;
console.log(number) //output 2

but if we try to do it in let it will give us an error. in let variable we can not redeclare a variable but we can reassign a value it is block scope so we can not access this value outside of function or any block. in const we can not redeclare or reassign a value and we can not access it's value outside of function it is also block scope.

What do you mean by REST API?

Rest API ( Representational State Transfer Application Programming Interface)
Rest API use HTTP protocol for data transfer. it is interact with client application and API server. A client send an HTTP request for data to an API Server and then server send an HTTP response to client application. Client also can POST,PUT,PATCH,DELETE data useing REST API.

Rest API Methods

  • GET = Get the data from server show to the client.
  • POST = Post data to the server.
  • PUT = Update or replace the data.
  • PATCH = Update exsisting the data.
  • DELETE = Delete the data from server.
